# Dashboard

## Visao Geral

O Dashboard e sua central de controle no Dumont Cloud. Aqui voce monitora maquinas, saldo, metricas e configura sua conta.

---

## Pagina Inicial

### Cards de Resumo
- **Maquinas Ativas**: Quantas GPUs estao rodando
- **Saldo Atual**: Creditos disponiveis
- **Gasto Hoje**: Quanto gastou nas ultimas 24h
- **Economia Total**: Quanto economizou vs AWS

### Grafico de Uso
Visualize o uso de GPU ao longo do tempo:
- Ultimas 24 horas
- Ultima semana
- Ultimo mes

---

## Menu Lateral

| Item | Descricao |
|------|-----------|
| **Dashboard** | Visao geral |
| **Machines** | Gerenciar GPUs |
| **GPU Metrics** | Metricas detalhadas |
| **Billing** | Creditos e pagamentos |
| **Spot Market** | Ofertas de GPU |
| **AI Wizard** | Assistente inteligente |
| **Settings** | Configuracoes |

---

## Barra Superior

### Notificacoes
Clique no sino para ver:
- Alertas de maquinas
- Avisos de saldo baixo
- Novidades do sistema

### Perfil
Clique no avatar para:
- Ver seu perfil
- Acessar configuracoes
- Fazer logout

### Busca Rapida
Use `Ctrl+K` para buscar:
- Maquinas por nome
- Documentacao
- Comandos rapidos

---

## Widgets

### Maquinas Recentes
Lista das ultimas 5 maquinas usadas com acesso rapido.

### Metricas de GPU
Grafico em tempo real do uso das suas GPUs ativas.

### Saldo e Previsao
Mostra saldo atual e previsao de quando acabara baseado no uso medio.

---

## Atalhos de Teclado

| Atalho | Acao |
|--------|------|
| `Ctrl+K` | Busca rapida |
| `Ctrl+N` | Nova maquina |
| `Ctrl+M` | Ir para Machines |
| `Ctrl+B` | Ir para Billing |
| `Ctrl+S` | Ir para Settings |
| `?` | Mostrar ajuda |

---

## Personalizacao

### Tema
1. Va em **Settings** > **Aparencia**
2. Escolha:
   - **Light**: Tema claro
   - **Dark**: Tema escuro
   - **System**: Segue o sistema

### Layout
Reorganize os widgets arrastando e soltando.

### Notificacoes
Configure quais notificacoes deseja receber:
- Email
- Push (browser)
- SMS (plano Pro+)
